004/**
005 * Enumeration of all possible types accessed by the Inspector API.
006 * Note that:
007 * - the EMPTY type is used as a placeholder where data is missing.
008 * - all integers are put into LONGs; the encoding takes care of
009 *   packing small integers compactly so this is also efficient.
010 * - likeweise DOUBLE is the only floating-point type, but "simple"
011 *   numbers (like 0.0 or 1.0) are packed compactly anyway.
012 * - DATA can be used anything for wrapping anything else serialized
013 *   as an array of bytes.
014 * - maps should be represented as an ARRAY of OBJECTs where each
015 *   object has the fields "key" and "value".
016 **/
017public enum Type {
018    EMPTY, BOOL, LONG, DOUBLE, STRING, DATA, ARRAY, OBJECT;
019}
