Requirements:

  • VirtualBox and Vagrant installed.

Run Vespa using Virtualbox and Vagrant

  1. Clone sample-apps repository

    $ git clone https://github.com/vespa-engine/sample-apps.git

  2. Start CentOS VM using Vagrant and Virtualbox

    Navigate into the cloned repository and start the CentOS VM using Vagrant. This may take a couple of minutes. 
    $ cd sample-apps
$ vagrant up

  3. SSH into the VM 

    $ vagrant ssh

  4. Deploy an sample application:

    The sample apps are located at /sample-apps inside the VM. 
    $ cd /sample-apps/basic-search
$ vespa-deploy prepare src/main/application && vespa-deploy activate

  5. Start Vespa:

    $ sudo service vespa start

  6. Check status Vespa:

    $ sudo service vespa status

  7. Feed documents : Feed sample documents: 

    $ java -jar $VESPA_HOME/lib/jars/vespa-http-client-jar-with-dependencies.jar --file music-data-feed.json --host localhost --port 8080

  8. Run a search query 

    $ curl -s http://localhost:8080/search/?query=bad |python -m json.tool
