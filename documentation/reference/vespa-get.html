This page provides the reference documentation for the vespa-get tool.

Name

vespa-get - Fetch documents from a Vespa content cluster.

Synopsis

vespa-get <options> [documentid...]

Description

vespa-get retrieves documents from a Vespa content cluster, and prints to stdout. vespa-get will retrieve documents identified by the document ids passed as command line arguments. If no document ids are passed through the command line interface, ids will be read from stdin - separated by line breaks.

Use Document field sets to configure fields returned.

Options

See the vespa-get help page (vespa-get --help) for a list of command line options.
